# Flint River Animal Hospital — SEO-Optimized Website

A highly optimized, SEO-focused static website for **Flint River Animal Hospital**, a full-service veterinary hospital located in Huntsville, AL.

## Pages

- **index.html** — Homepage with hero, services overview, testimonials, FAQ (with FAQPage schema)
- **about.html** — About the hospital, history, awards, and full veterinary team bios
- **services.html** — Detailed pages for all veterinary services (wellness, surgery, dental, vaccinations, diagnostics, emergency, microchipping, pharmacy, grooming)
- **boarding.html** — Bed 'n Biscuit boarding facility and Paws 2 Play doggie daycare
- **contact.html** — Contact info, hours, embedded map, and contact form

## SEO Features

- **Semantic HTML5** structure with proper heading hierarchy
- **Schema.org JSON-LD** structured data (VeterinaryCare, FAQPage, BreadcrumbList, LocalBusiness, LodgingBusiness)
- **Open Graph & Twitter Card** meta tags on every page
- **Canonical URLs** on all pages
- **XML Sitemap** and **robots.txt**
- **Breadcrumb navigation** with structured data
- **FAQ accordion** with FAQPage schema markup
- **Mobile-responsive** design
- **Accessible** — ARIA labels, skip links, semantic landmarks, proper alt text
- **Performance optimized** — No external font loading, minimal CSS, no JS frameworks
- **Local SEO** — NAP consistency, geo meta tags, Google Maps embed
